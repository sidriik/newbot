HSE BookBot – Telegram бот для управления личной библиотекой
HSE BookBot – это умный Telegram-бот для управления личной библиотекой. Он помогает отслеживать прогресс чтения, оценивать книги, получать рекомендации и вести статистику прочитанного.

Возможности:
Основные функции
- Управление личной библиотекой – добавляйте, удаляйте и организуйте книги
- Отслеживание прогресса – обновляйте текущую страницу, видите процент прочтения
- Система оценок – ставьте оценки от 1 до 5 звезд прочитанным книгам
- Умный поиск – ищите книги по названию, автору или жанру
- Топ книги – просматривайте самые популярные и высокооцененные книги
- Статистика – анализируйте свои читательские привычки

 Статусы чтения
- Запланировано – книги, которые вы планируете прочитать
- Читаю сейчас – книги в процессе чтения с отслеживанием прогресса
- Прочитано – завершенные книги с возможностью оценки
- Брошено – книги, которые вы решили не дочитывать

Аналитика:
- Общее количество книг в библиотеке
- Количество книг по каждому статусу
- Средняя оценка ваших книг
- Прогресс чтения в реальном времени
